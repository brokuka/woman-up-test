# Список задач

> Простенький проект списка задач на ReactJS + Firebase.

## Таблица контента

1.  [Установка](#1-installation)
2.  [Запуск](#2-running)

## 1. Установка

Установка зависимостей проекта

```sh
npm install
```

## 2. Запуск

Запуска в `development` режиме

```sh
npm run dev
```

## 3. Build

Запуск сборки проекта

```sh
npm run build
```
